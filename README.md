# 🎥 Video Meeting Application

A modern, real-time video conferencing solution built with React and WebRTC.

## ✨ Features

- 📹 High-quality video and audio calls
- 💬 Real-time chat messaging during calls
- 🔗 Easy meeting sharing via links
- 🌙 Light/Dark mode support
- 👥 Multiple participant support
- 🔐 Secure, encrypted connections
- 🛠️ Screen sharing capabilities
- 🎨 Modern, responsive design

## 🚀 Getting Started

### Prerequisites

- Node.js 16.x or later
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Naziz2/video-meetig.git
cd video-meetig
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open your browser and navigate to `http://localhost:3000`

## 🔧 Technology Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **State Management**: React Context API
- **Styling**: Tailwind CSS with custom components
- **Video/Audio**: WebRTC
- **Notifications**: React Hot Toast
- **UI Components**: Custom components with responsive design

## 📖 Usage

### Creating a Meeting

1. Navigate to the homepage
2. Click on "Start a Meeting" button
3. Grant camera and microphone permissions when prompted
4. Share the meeting link with participants

### Joining a Meeting

1. Use a meeting link shared with you
2. Enter your name when prompted
3. Grant camera and microphone permissions
4. Click "Join Meeting"

### In-Meeting Controls

- 🎤 Toggle microphone on/off
- 📹 Toggle camera on/off
- 📱 Switch camera (if multiple available)
- 💻 Share your screen
- 💬 Open chat panel
- 🔴 End call or leave meeting
